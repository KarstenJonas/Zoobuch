Schritte zum Erstellen eines Angular-Projekts

  1. Node.js und npm installieren
  2. Angular CLI installieren: npm install -g @angular/cli
  3. Neues Projekt erstellen: ng new projektname
  4. Um Modules zu verwenden: ng new projektname --no-standalone
  5. In das Projektverzeichnis wechseln: cd projektname
  6. Entwicklungsserver starten: ng serve
